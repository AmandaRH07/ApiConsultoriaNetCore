# ApiConsultoriaNetCore
CRUD de uma API de consultoria médica. Usuários realizam o login e são divididos em Clientes e Médicos.

## Tabela de conteúdos
- Fluent Validation
- AutoMapper
- Swagger
- Logs
- Entity Framework Core
- JWT
- Testes de unidade:
    - Fluent Assertions
    - Bogus
    - NSubstitute

## Features
- Login de Usuários
- Cadastro de Clientes
- Cadastro de Médicos

## Tecnologia
- C# 
- .Net Core 3.1
